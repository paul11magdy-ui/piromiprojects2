PIROMI PROJECTS — GITHUB PAGES SETUP

IMPORTANT: Do not upload this ZIP file directly into the repository.

1. Extract this ZIP on your computer.
2. Open the extracted folder.
3. In your GitHub repository, choose Add file > Upload files.
4. Drag ALL of these extracted items into GitHub:
   - index.html
   - projects.html
   - assets folder
   - projects-assets folder
   - .nojekyll
5. Commit the files to the main branch.
6. Open Settings > Pages.
7. Under Build and deployment:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: /(root)
8. Click Save.
9. Wait a few minutes, then refresh the Pages section for the website address.

Check that index.html appears directly on the repository's main page.
It must not be inside another folder and must not remain inside the ZIP.

Note: The website will display on GitHub Pages, but the Netlify quote form
will only process submissions when the repository is connected to Netlify.
